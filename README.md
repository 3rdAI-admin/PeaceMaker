# PeaceMaker — Therapeutic Puzzle

A slow, calming falling-block puzzle designed for extended therapeutic play sessions.

## The Science

PeaceMaker is based on the **Imagery Competing Task Intervention (ICTI)** — research from Oxford and Cambridge showing that visuospatial tasks like mental rotation of geometric shapes can act as a "cognitive vaccine" against traumatic memories.

**Key findings:**
- Engaging in visuospatial tasks during memory reconsolidation disrupts the formation of vivid traumatic flashbacks
- Oxford study (2026): patients reported **10x fewer flashbacks** within 4 weeks; **70% had zero flashbacks** after 6 months
- The mechanism works by occupying the brain's visual memory circuits, competing with traumatic imagery
- Most effective when played shortly after a traumatic event or therapy session (EMDR)

**Sources:**
- [Oxford study: Digital treatment with Tetris gameplay dramatically reduces trauma memories](https://www.ox.ac.uk/news/2026-02-19-study-shows-digital-treatment-tetris-gameplay-can-dramatically-reduce-trauma)
- [Cambridge: Tetris gameplay treatment helps reduce traumatic flashbacks](https://www.cam.ac.uk/research/news/tetris-gameplay-treatment-helps-reduce-traumatic-flashbacks-for-frontline-healthcare-workers)
- [Broughill 2025: Dose-response effect of visuospatial game on intrusive memories](https://onlinelibrary.wiley.com/doi/10.1002/jts.70000)

## Therapeutic Design Principles

| Principle | Implementation |
|-----------|---------------|
| **Slow pace** | 2-second drop speed, gets slower over time |
| **Visuospatial engagement** | Mental rotation of geometric shapes |
| **No punishment** | No game over — if board fills, top rows clear compassionately |
| **No scoring** | Session timer + rows cleared only (no competitive pressure) |
| **Breathing guide** | 4-phase breathing animation (inhale, hold, exhale, rest) |
| **Gentle affirmations** | Rotating calming messages |
| **Extended play** | Designed for 1-hour+ sessions |
| **Calming aesthetics** | Pastel colors, dark background, soft glow effects |

## Legal Distinction

PeaceMaker is **not** Tetris. It is legally distinct:
- **Board:** 10x18 (not 10x20)
- **Shapes:** Mix of 3, 4, and 5-cell pieces (not standard tetrominoes)
- **Rotation:** Bounding-box center rotation (not Super Rotation System)
- **No game over:** Compassionate row clearing instead
- **No scoring:** No points, levels, or competitive elements
- **Name:** "PeaceMaker" — no Tetris branding

## How to Play

Open `index.html` in any browser. No server required.

- **← →** Move left/right
- **↑** Rotate shape
- **↓** Nudge down
- **Space** Place immediately
- **Touch:** Tap to rotate, swipe to move

## Note

While PeaceMaker shows promise as a supplementary tool for managing intrusive memories, it is **not a replacement for professional therapy**. Best used under clinical guidance or as a complementary coping strategy.
